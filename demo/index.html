About OxiGDAL

OxiGDAL is a Pure Rust implementation of geospatial data processing libraries, providing efficient COG (Cloud Optimized GeoTIFF) reading capabilities in the browser via WebAssembly. This demo showcases range request support, tile-based rendering, and metadata extraction without requiring any backend services.
